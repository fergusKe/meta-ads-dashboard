# 耘初茶食 Meta 廣告分析儀表板

一個全面的 Meta 廣告數據分析與 AI 創意生成平台，專為耘初茶食打造。

## 🚀 快速開始

### 環境需求
- Python 3.10+
- uv (Python 包管理器)

### 安裝與啟動

1. **克隆專案並進入目錄**
```bash
cd meta-ads-dashboard
```

2. **使用 uv 安裝依賴**
```bash
uv sync
```

3. **設定環境變數**
建立 `.env` 檔案並設定以下變數：
```env
OPENAI_API_KEY=your_openai_api_key
GEMINI_API_KEY=your_gemini_api_key
DATA_FILE_PATH=耘初茶食.xlsx
```

4. **確保數據檔案存在**
將 `耘初茶食.xlsx` 檔案放在專案根目錄

5. **啟動應用**
```bash
uv run streamlit run app.py
```

應用將在 http://localhost:8501 啟動

## 📊 主要功能

### 1. 🏠 首頁概覽
- **廣告效能概覽**：總花費、ROAS、轉換次數等關鍵指標

- **趨勢圖表**：ROAS、花費、轉換趨勢分析
- **日期範圍篩選**：根據實際廣告投放期間篩選數據

### 2. 📊 整體效能儀表板
- **關鍵指標監控**：實時顯示重要 KPI
- **效能趨勢分析**：多維度趨勢圖表
- **漏斗分析**：從曝光到轉換的完整路徑
- **效能對比**：不同時期、活動間的效能比較

### 3. 🎯 活動分析

### 5. 💰 ROI 分析

- **成本效益分析**：CPA、CPM、CPC 等成本指標

### 6. 📈 趨勢分析
- **高級統計分析**：時間序列分析、季節性檢測
- **預測模型**：未來趨勢預測
- **異常檢測**：自動識別數據異常
- **變化點分析**：關鍵轉折點識別

### 7. ⚡ 即時優化建議

## 🔧 技術架構

### 核心技術
- **前端框架**：Streamlit
- **數據處理**：Pandas, NumPy
- **視覺化**：Plotly, Plotly Express
- **AI 整合**：
  - OpenAI GPT-5-nano (文案生成)
  - Gemini nano-banana (圖片生成)

- **支援欄位**：46 個廣告指標欄位
- **數據範圍**：完整的廣告投放歷史數據

## 📊 數據格式

### 支援的 Excel 欄位
- **活動資訊**：行銷活動名稱、廣告組合名稱、廣告名稱
- **投放設定**：目標、年齡、性別、投遞狀態
- **時間資訊**：開始、結束時間、分析報告開始/結束
- **效能指標**：觸及人數、曝光次數、點擊次數、花費金額
- **轉換數據**：購買次數、購買 ROAS、每次購買成本
- **其他指標**：CTR、CPM、CPC、頻率等

## ⚙️ 配置說明

### 環境變數
- `OPENAI_API_KEY`：OpenAI API 金鑰（文案生成功能）
- `GEMINI_API_KEY`：Gemini API 金鑰（圖片生成功能）
- `DATA_FILE_PATH`：數據檔案路徑（預設：耘初茶食.xlsx）

### 功能開關
- AI 功能需要對應的 API 金鑰才能使用
- 數據分析功能不依賴外部 API，可獨立使用

## 🔒 安全注意事項

1. **API 金鑰保護**：確保 `.env` 檔案不被提交到版本控制
2. **數據隱私**：廣告數據僅在本地處理，不會上傳到外部服務
3. **存取控制**：建議在安全環境中運行應用

## 🚀 使用建議

### 最佳實踐
1. **定期更新數據**：保持 Excel 檔案為最新版本
2. **監控關鍵指標**：重點關注 ROAS、CPA、CTR 等核心指標
3. **結合 AI 建議**：善用 AI 生成的優化建議
4. **A/B 測試**：持續測試不同創意和策略
5. **數據備份**：定期備份分析結果和優化記錄

### 故障排除
- **數據載入失敗**：檢查 Excel 檔案路徑和格式
- **API 錯誤**：確認 API 金鑰設定正確
- **圖表顯示異常**：檢查數據範圍和篩選條件
- **效能問題**：考慮縮小數據範圍或清除快取
